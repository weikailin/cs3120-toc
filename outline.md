Course outline (updated Mar 2, 2026):
- Sets and Cardinality. (4-5 classes)
  - Class 0: Introduction. ---[&#x1F4D6;Barak, Sec 0](https://introtcs.org/public/lec_01_introduction.html).
  - Class 1: Definitions. ---[&#x1F4D6;Barak, Sec 1](https://introtcs.org/public/lec_00_1_math_background.html).
  - Class 2: Cardinality. ---[&#x1F4D6;Barak, Sec 2](https://introtcs.org/public/lec_02_representation.html).
  - Class 3: Infinity. ---[&#x1F4D6;Barak, Sec 2](https://introtcs.org/public/lec_02_representation.html).
  - Class 4: Cantor's theorem. Finite Automata. --- [&#x1F4D6;Barak, Sec 2](https://introtcs.org/public/lec_02_representation.html), [&#x1F4D6;Barak, Sec 6](https://introtcs.org/public/lec_05_infinite.html).

- Regular Functions. (5-6 classes)
  - Class 5: Regular Expressions. --- [&#x1F4D6;Barak, Sec 6](https://introtcs.org/public/lec_05_infinite.html).
  - Class 6: Regular Expressions vs DFA --- [&#x1F4D6;Barak, Sec 6](https://introtcs.org/public/lec_05_infinite.html). 
  - Class 7: DFA to Regular Expressions. --- [&#x1F4D6;Barak, Sec 6.4.2](https://introtcs.org/public/lec_05_infinite.html#regdfaequivsec)
  - Class 8: Non-deterministic Finite Automata. --- [&#x1F4D6;Sipser, Sec 1.2]
  - Class 9: NFA to DFA. --- [&#x1F4D6;Sipser, Introduction to the Theory of Computation, Sec 1.2-1.3, Theorem 1.39 and "closure under the regular operations"](https://search.lib.virginia.edu/sources/uva_library/items/u2911801)

- Circuits. (4-5 classes)
  - Class 10: Universal gates. --- [&#x1F4D6;Barak, Sec 4.6](https://introtcs.org/public/lec_03a_computing_every_function.html).
  - Class 11: Non-universal gates, complexity. --- [&#x1F4D6;Barak, Sec 4.6](https://introtcs.org/public/lec_03a_computing_every_function.html).
  - Class 12: Program as Data. --- [&#x1F4D6;Barak, Sec 5](https://introtcs.org/public/lec_04_code_and_data.html).
  - Class 13: Circuit Size Hierarchy. --- [&#x1F4D6;Barak, Sec 5.2.1](https://introtcs.org/public/lec_04_code_and_data.html#size-hierarchy-theorem-optional).
